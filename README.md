# Eye Care Timer

A simple 20-20-20 eye care timer built with Vue.

It reminds you to take a short break after focusing on the screen for a while to help reduce eye strain.

You can adjust the focus time and rest time, and the page will show a full-screen reminder when it's time to rest.

The timer runs entirely in the browser and does not require any backend.
